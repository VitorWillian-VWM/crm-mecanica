# 🛠️ CRM Mecânica - Precision System

Sistema CRM para gestão de oficinas mecânicas.

## 🚀 Funcionalidades

- Cadastro de clientes
- Controle de veículos
- Busca por nome, telefone e placa
- Modal de cadastro
- Atualização automática (quase tempo real)
- Estrutura preparada para:
  - Ordem de Serviço (O.S)
  - Estoque
  - Financeiro

## 🧠 Tecnologias

- HTML
- CSS
- JavaScript
- PHP (API)
- JSON (Banco de dados)
